To report an error or submit a request, please include the following details in your email:

  • Project: Specify the project name (e.g., My IP).
  • Error Description: Detail what happened and how to reproduce it.
  • Environment: Indicate the operating system, browser, or device used.
  • Attachments: If possible, attach screenshots or videos.

Send your email to tatamiweb@proton.me

[ What Are Browser Extensions? ]


Extensions are small tools that run inside your browser (like Chrome or Firefox) to add or improve features. Examples include ad blockers, password managers, or privacy tools.
